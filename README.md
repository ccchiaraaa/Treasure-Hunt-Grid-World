# Treasure-Hunt-Grid-World
A reinforcement learning game where an AI agent learns to navigate a 6x6 grid world, avoid traps, and find treasure using the Q-learning algorithm. Built with Python and Pygame, this project demonstrates how an agent can learn optimal behavior through trial and error, with real-time visualization of the learning process. Features include human control mode, AI training mode with adjustable episodes, ε-greedy exploration, and comprehensive statistics display. Watch as the blue agent starts by randomly moving into traps and gradually learns to find the gold at position (3,3) for a +10 reward while avoiding -5 trap penalties.
